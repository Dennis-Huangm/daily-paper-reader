## 摘要
企业在采用大语言模型（LLMs）时，受到幻觉、领域漂移以及无法在推理层面强制执行监管合规的限制。我们提出了一种在 Foundation AgenticOS (FAOS) 平台中实现的神经符号架构，通过本体约束的神经推理来解决这些局限性。我们的方法引入了一个三层本体框架——角色本体、领域本体和交互本体——为基于 LLM 的企业级智能体提供形式化语义落地。我们形式化了非对称神经符号耦合的概念，其中符号化本体知识约束智能体的输入（上下文组装、工具发现、治理阈值），同时提出了将此耦合扩展到约束智能体输出（响应验证、推理校验、合规检查）的机制。我们通过一项受控实验（在金融科技、保险、医疗、越南银行业和越南保险业五个行业中进行了 600 次运行）对该架构进行了评估，结果发现本体耦合智能体在指标准确性（p < .001, W = .460）、监管合规性（p = .003, W = .318）和角色一致性（p < .001, W = .614）方面显著优于非落地智能体，且在 LLM 参数化知识最薄弱的领域（特别是越南本地化领域）提升最为显著。我们的贡献包括：(1) 一种形式化的三层企业本体模型；(2) 神经符号耦合模式的分类法；(3) 通过 SQL 下推评分实现的本体约束工具发现；(4) 一种提出的输出端本体验证框架；(5) 逆参数化知识效应的实证证据，即本体落地价值与 LLM 训练数据对该领域的覆盖率成反比；(6) 一个服务于 21 个垂直行业、拥有 650 多个智能体的生产系统。

## Abstract
Enterprise adoption of Large Language Models (LLMs) is constrained by hallucination, domain drift, and the inability to enforce regulatory compliance at the reasoning level. We present a neurosymbolic architecture implemented within the Foundation AgenticOS (FAOS) platform that addresses these limitations through ontology-constrained neural reasoning. Our approach introduces a three-layer ontological framework--Role, Domain, and Interaction ontologies--that provides formal semantic grounding for LLM-based enterprise agents. We formalize the concept of asymmetric neurosymbolic coupling, wherein symbolic ontological knowledge constrains agent inputs (context assembly, tool discovery, governance thresholds) while proposing mechanisms for extending this coupling to constrain agent outputs (response validation, reasoning verification, compliance checking). We evaluate the architecture through a controlled experiment (600 runs across five industries: FinTech, Insurance, Healthcare, Vietnamese Banking, and Vietnamese Insurance), finding that ontology-coupled agents significantly outperform ungrounded agents on Metric Accuracy (p < .001, W = .460), Regulatory Compliance (p = .003, W = .318), and Role Consistency (p < .001, W = .614), with improvements greatest where LLM parametric knowledge is weakest--particularly in Vietnam-localized domains. Our contributions include: (1) a formal three-layer enterprise ontology model, (2) a taxonomy of neurosymbolic coupling patterns, (3) ontology-constrained tool discovery via SQL-pushdown scoring, (4) a proposed framework for output-side ontological validation, (5) empirical evidence for the inverse parametric knowledge effect that ontological grounding value is inversely proportional to LLM training data coverage of the domain, and (6) a production system serving 21 industry verticals with 650+ agents.

#### 2. 方法论
论文提出了基于 **Foundation AgenticOS (FAOS)** 平台的神经符号架构，核心思想是利用三层本体框架对智能体进行“非对称耦合”约束：
*   **三层企业本体模型 (O = ⟨R, D, I⟩)：**
    *   **角色本体 (Role, R)：** 定义特定职位的决策模式、KPI 关注点、沟通风格和审批权限。
    *   **领域本体 (Domain, D)：** 包含行业层级结构、实体定义、度量指标（含健康范围）及监管约束。
    *   **交互本体 (Interaction, I)：** 形式化组织内的工作流、角色间的移交协议（Handoff）及升级路径。
*   **关键技术细节：**
    *   **输入端耦合：** 通过 `ContextResolver` 将符号化定义转化为自然语言注入上下文；利用 SQL 下推（SQL-pushdown）技术，根据领域层级对工具（Skills）进行语义发现和过滤。
    *   **治理阈值：** 针对受监管行业设置质量门禁，只有达到特定治理分数的技能才会被调用。
    *   **输出端验证（提案）：** 提出 `OntologyValidator` 概念，通过描述逻辑（OWL 推理器）检查 LLM 输出是否违反本体定义的指标范围或监管条款。

#### 3. 实验设计
*   **场景与数据集：** 涵盖 5 个受监管行业：金融科技、保险、医疗、越南银行业、越南保险业。共设计了 50 个评估任务（每行业 10 个），涵盖术语、指标解释、合规性、角色分析等维度。
*   **对比方法（Benchmark）：**

#### 5. 实验数量与充分性
*   **实验规模：** 进行了 **600 次**受控实验运行（50 个任务 × 4 种条件 × 3 次重复）。

#### 6 主要结论与发现
*   **显著提升：** 本体耦合智能体在指标准确性（MA）、监管合规性（RC）和角色一致性（RS）上显著优于非落地智能体（p < .001）。
*   **逆参数化知识效应（Inverse Parametric Knowledge Effect）：** 本体落地的价值与 LLM 预训练数据对该领域的覆盖率成反比。在 LLM 知识最薄弱的越南本地化领域，性能提升幅度是英语领域的 2 倍。
*   **上下文干扰：** 对于 LLM 已经掌握得很好的通用概念，注入本体上下文反而可能导致性能轻微下降（干扰了模型的原生参数知识）。
*   **角色一致性：** 角色特定的决策模式在不同语言间表现出高度的一致性提升。

#### 7. 优点
*   **架构严谨：** 提出了清晰的三层本体模型，将抽象的企业管理逻辑转化为可计算的约束。
*   **实战导向：** 架构已在生产环境中落地，支持 21 个垂直行业和 650 多个智能体，具有极强的工程参考价值。
*   **多语言洞察：** 深入探讨了 LLM 在小语种/特定国家监管环境下的知识短板，并提供了有效的补齐方案。
*   **高效发现：** 采用 SQL 下推评分机制解决了大规模工具库（600+ 技能）的快速检索问题。

#### 8. 不足与局限
*   **非对称性：** 目前生产系统主要实现了输入端约束，输出端的实时本体验证（L4-L5 级别）仍处于架构提案阶段，尚未完全实现。
*   **维护成本：** 本体工程需要专家维护，随着监管政策的快速变动，本体的更新频率和成本可能成为瓶颈。
*   **推理延迟：** 引入复杂的 OWL 推理器可能会增加交互式场景下的响应延迟。
*   **裁判偏差：** 依赖 LLM 作为裁判可能存在主观性，虽然与人类评估有相关性，但仍需更多人类专家验证。
