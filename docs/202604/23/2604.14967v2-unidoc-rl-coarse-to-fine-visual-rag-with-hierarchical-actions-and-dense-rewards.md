## 摘要
检索增强生成 (RAG) 通过外部视觉知识扩展了大视觉语言模型 (LVLM)。然而，现有的视觉 RAG 系统通常依赖于通用的检索信号，忽略了复杂推理所必需的细粒度视觉语义。为了解决这一局限性，我们提出了 UniDoc-RL，这是一个统一的强化学习框架，其中 LVLM 智能体共同执行检索、重排序、主动视觉感知和推理。UniDoc-RL 将视觉信息获取建模为一个具有分层动作空间的序列决策问题。具体而言，它将视觉证据从粗粒度的文档检索逐步细化为细粒度的图像选择和主动区域裁剪，使模型能够抑制无关内容并关注信息密集区域。为了实现有效的端到端训练，我们引入了一种稠密多奖励方案，为每个动作提供任务感知的监督。基于群体相对策略优化 (GRPO)，UniDoc-RL 在不依赖独立价值网络的情况下，使智能体行为与多个目标保持一致。为了支持这种训练范式，我们构建了一个包含细粒度动作标注的高质量推理轨迹综合数据集。在三个基准测试上的实验表明，UniDoc-RL 持续超越了最先进的基准模型，相比之前的基于强化学习的方法提升了高达 17.7%。

## Abstract
Retrieval-Augmented Generation (RAG) extends Large Vision-Language Models (LVLMs) with external visual knowledge. However, existing visual RAG systems typically rely on generic retrieval signals that overlook the fine-grained visual semantics essential for complex reasoning. To address this limitation, we propose UniDoc-RL, a unified reinforcement learning framework in which an LVLM agent jointly performs retrieval, reranking, active visual perception, and reasoning. UniDoc-RL formulates visual information acquisition as a sequential decision-making problem with a hierarchical action space. Specifically, it progressively refines visual evidence from coarse-grained document retrieval to fine-grained image selection and active region cropping, allowing the model to suppress irrelevant content and attend to information-dense regions. For effective end-to-end training, we introduce a dense multi-reward scheme that provides task-aware supervision for each action. Based on Group Relative Policy Optimization (GRPO), UniDoc-RL aligns agent behavior with multiple objectives without relying on a separate value network. To support this training paradigm, we curate a comprehensive dataset of high-quality reasoning trajectories with fine-grained action annotations. Experiments on three benchmarks demonstrate that UniDoc-RL consistently surpasses state-of-the-art baselines, yielding up to 17.7% gains over prior RL-based methods.

*   **核心问题**：现有的视觉检索增强生成（Visual RAG）系统主要依赖通用的检索信号，难以捕捉复杂推理所需的细粒度视觉语义。
*   **研究背景**：视觉文档（如图表、扫描报告）包含大量冗余背景噪声。现有的方法通常将检索与推理分离，或者仅采用被动的视觉消费模式（直接编码整图），导致模型容易受到无关信息干扰，且在优化过程中面临“信用分配”（Credit Assignment）难题，即难以确定最终答案的对错归因于检索、裁剪还是推理步骤。

### 2. 方法论
UniDoc-RL 将视觉 RAG 建模为一个**序列决策问题**，核心包含以下三个维度：
*   **分层动作空间（Hierarchical Action Space）**：
    1.  **图像搜索（Search）**：调用外部工具进行粗粒度候选文档检索。
    2.  **精确选择（Select）**：利用 LVLM 智能体对候选图进行语义重排序和筛选，过滤文档级噪声。
    3.  **主动视觉感知（Perceive）**：通过 `<bbox>` 动作主动定位、裁剪并放大关键区域（ROI），模拟人类视觉注意力。
*   **稠密多奖励机制（Dense Multi-Rewards）**：
    *   **检索奖励**：基于 NDCG 评估搜索查询的质量。
    *   **选择奖励**：评估所选图像是否包含真实证据（Ground Truth）。
    *   **感知奖励**：计算预测裁剪框与真实区域的交并比（IoU）。
    *   **格式奖励**：确保模型遵循预定义的 XML 标签格式。
    *   **结果奖励**：由奖励模型（RM）评估最终答案的正确性。
*   **训练算法**：采用**群体相对策略优化（GRPO）**。该算法通过组内样本的相对得分来估计优势函数，无需独立的价值网络（Value Network），显著降低了显存开销。
*   **数据合成**：利用 Qwen3-VL-235B 作为教师模型，结合 Mineru 布局分析工具，自动合成了包含 1.2 万条 SFT 轨迹和 5.5 千条 RL 轨迹的高质量数据集。

*   **对比基准（Baselines）**：

### 5. 实验数量与充分性
*   **实验规模**：论文在 3 个主流基准上进行了全面测试，涵盖了单跳、多跳、逻辑推理及多种视觉类型（文本、表格、图表、布局）。
*   **消融实验**：针对“分层动作”和“多奖励设计”分别做了消融，验证了“选择”和“裁剪”动作对不同任务的贡献度。
*   **定性分析**：统计了 RL 训练前后模型调用裁剪动作的频率变化，并展示了具体案例。
*   **评价**：实验设计客观、公平，对比了 OCR 和纯视觉两种路径，且在相同 Prompt 设定下进行对比，具有较强的说服力。

### 6. 主要结论与发现
*   **性能飞跃**：UniDoc-RL 在 3B 和 7B 模型上均刷新了 SFT 和 RL 基准，相比之前的 VRAG-RL 提升了约 **17.5% - 17.7%**。
*   **RL 的作用**：强化学习不仅提升了最终准确率，还纠正了 SFT 模型的“暴露偏差”，使模型从被动读图转变为主动寻找关键信息（裁剪频率显著提升）。
*   **降噪有效性**：通过“选择”动作，检索召回率在各个数据集上均有显著提升，证明了 LVLM 作为重排序器的潜力。

### 7. 优点
*   **统一性**：将检索、重排序、感知和推理整合进单一的 RL 决策流中，实现了端到端优化。
*   **由粗到精**：分层动作设计非常符合人类处理复杂文档的逻辑（先找书，再翻页，最后盯着图表看）。
*   **稠密监督**：解决了长链条推理中的奖励稀疏问题，使中间步骤的优化更加明确。

### 8. 不足与局限
*   **数据依赖**：高质量轨迹的合成高度依赖于极强大的教师模型（如 Qwen3-VL-235B），对于缺乏此类资源的研究者较难复现数据生成过程。
*   **推理成本**：多轮迭代（搜索-选择-裁剪-回答）虽然提升了准确性，但也增加了推理时的 Token 消耗和时间延迟。
*   **场景局限**：目前主要集中在文档类 RAG，对于自然场景下的开放域视觉 RAG（如视频或实景地图）的泛化能力尚待验证。
