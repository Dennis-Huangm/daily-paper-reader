## 摘要
最近，隐空间推理（latent reasoning）被引入大语言模型（LLMs），以利用连续空间中的丰富信息。然而，由于缺乏随机采样，这些方法不可避免地会退化为确定性推理，从而无法发现多样化的推理路径。为了弥补这一差距，我们通过 Gumbel-Softmax 为隐空间推理注入了可控的随机性，恢复了 LLMs 的探索能力，并增强了其与强化学习（RL）的兼容性。在此基础上，我们提出了隐空间推理策略优化（LEPO），这是一个直接在连续隐表征上应用强化学习的新型框架。具体而言，在展开（rollout）阶段，LEPO 保持随机性以实现多样化的轨迹采样；而在优化阶段，LEPO 为隐表征和离散 Token 构建了统一的梯度估计。广泛的实验表明，LEPO 在离散推理和隐空间推理方面均显著优于现有的强化学习方法。

## Abstract
Recently, latent reasoning has been introduced into large language models (LLMs) to leverage rich information within a continuous space. However, without stochastic sampling, these methods inevitably collapse to deterministic inference, failing to discover diverse reasoning paths. To bridge the gap, we inject controllable stochasticity into latent reasoning via Gumbel-Softmax, restoring LLMs' exploratory capacity and enhancing their compatibility with Reinforcement Learning (RL). Building on this, we propose \textbf{\underline{L}}atent R\textbf{\underline{e}}asoning \textbf{\underline{P}}olicy \textbf{\underline{O}}ptimization~(\textbf{LEPO}), a novel framework that applies RL directly to continuous latent representations. Specifically, in rollout stage, LEPO maintains stochasticity to enable diverse trajectory sampling, while in optimization stage, LEPO constructs a unified gradient estimation for both latent representations and discrete tokens. Extensive experiments show that LEPO significantly outperforms existing RL methods for discrete and latent reasoning.

### 1. 核心问题与研究动机
*   **核心问题**：现有的隐空间推理（Latent Reasoning）方法虽然能利用连续空间中的丰富信息，但大多是**确定性推理**，缺乏随机采样机制。
*   **研究动机**：在强化学习（RL）中，探索（Exploration）至关重要。确定性推理导致模型无法发现多样化的推理路径（轨迹级探索缺失），限制了模型在复杂任务中的性能。作者旨在为隐空间推理引入可控随机性，并构建一套直接优化连续表征的强化学习框架。

### 2. 方法论
LEPO 框架主要包含两个阶段：
*   **核心思想**：通过 Gumbel-Softmax 为隐表征注入随机噪声，恢复模型的探索能力，并利用统一的策略梯度目标同时优化隐 Token 和离散 Token。
*   **关键技术细节**：
    *   **随机采样（Rollout）**：在推理步骤中，不直接取概率最大的 Token，而是对词表分布 $\pi_t$ 注入 Gumbel 噪声，通过 Gumbel-Softmax 得到随机隐 Token $z_t$。这使得模型能生成多样化的隐推理轨迹。
    *   **混合轨迹生成**：模型先生成 $L_g$ 个隐 Token 进行“思考”，随后切换到离散采样生成可读的最终答案。
    *   **统一优化目标**：
        *   **离散部分**：对答案部分的 Token 使用标准的 REINFORCE 算法。
        *   **隐空间部分**：将采样的随机向量 $z_t$ 视为“软标签”，通过最大化当前策略与该软标签的对齐度（加权优势函数 $A_i$）来进行优化。
    *   **优势函数计算**：采用类似 GRPO 的组内奖励归一化方法，无需额外的 Critic 网络。

### 3. 实验设计
*   **基础模型**：Qwen2.5 (7B, 3B) 和 Llama-3.2-3B-Instruct。
*   **数据集**：
    *   **数学推理（核心）**：GSM8K, MATH500, MinervaMath, AIME2024, AIME2025, AMC23。
    *   **分布外（OOD）泛化**：GPQA-Diamond, ARC-C, MMLU-STEM。
*   **对比方法**：
    *   **离散基准**：CoT（思维链）、GRPO（离散强化学习）。
    *   **隐空间基准**：Soft Tokens、HRPO（现有的隐空间 RL 方法）、Coconut/CODI（基于 SFT 的隐空间方法）。

### 4. 资源与算力
*   **硬件**：使用了 **8 台 Nvidia H20 GPU**。
*   **训练时长**：
    *   Qwen2.5-7B：约 **28 小时**。

### 5. 实验数量与充分性
*   **实验规模**：论文涵盖了 9 个主流基准测试，并在 3 种不同规模和架构的模型上进行了验证。
*   **消融实验**：非常充分。作者研究了隐推理长度（$L_g$）、Gumbel 温度（$\tau_g$）、噪声类型（对比了高斯噪声和狄利克雷噪声）对性能的影响。
*   **客观性**：通过 Pass@1 和 Pass@32 两个指标评估性能和探索覆盖率，并提供了训练过程中的熵（Entropy）变化和 Token 消耗曲线，证明了实验结果的可靠性。

### 6. 主要结论与发现
*   **性能提升**：LEPO 在所有基准测试中均优于离散 RL（GRPO）和现有的隐空间 RL 方法。在 Qwen2.5-7B 上，平均 Pass@1 提升了 1.76%，Pass@32 提升了 1.95%。
*   **探索能力**：引入 Gumbel 噪声显著提高了训练过程中的策略熵，使模型能将“不可解”的难题转化为“可解”的中等难度题目。
*   **效率优势**：相比离散 CoT，LEPO 在达到同等或更高精度时，消耗的 Token 数量更少，推理更高效。
*   **收敛性**：LEPO 的训练奖励曲线比使用高斯噪声的方法更平稳且收敛更快。

### 7. 优点
*   **理论完备性**：首次将 Gumbel-Softmax 引入隐空间推理 RL，解决了连续表征下难以进行轨迹探索的痛点。
*   **统一性**：提出的优化目标完美兼容了软目标（隐空间）和硬目标（离散空间），实现了端到端的优化。
*   **实用性**：在提升推理能力的同时降低了 Token 消耗，具有实际部署的潜力。

### 8. 不足与局限
*   **可解释性**：隐空间推理步骤对人类不可读，虽然可以通过解码 Top-1 Token 近似理解，但仍不如离散 CoT 直观。
*   **生成范式限制**：目前采用“先隐后显”的顺序结构，不支持隐 Token 与离散 Token 的动态交替（即边想边写），这可能限制了其在某些开放式任务中的灵活性。
*   **超参数敏感性**：隐空间长度和温度对结果有一定影响，可能需要针对不同模型进行微调。
