## 摘要
大语言模型（LLMs）日益依赖智能体能力（如迭代检索、工具使用和决策）来克服静态参数化知识的局限性。然而，现有的智能体框架将外部信息视为非结构化文本，未能利用现实世界数据中固有的拓扑依赖关系。为了填补这一空白，我们引入了智能体图学习（Agentic Graph Learning, AGL），这一范式将图学习重新定义为拓扑感知导航与基于 LLM 推理的交替过程。具体而言，我们提出了 AgentGL，这是首个由强化学习（RL）驱动的 AGL 框架。AgentGL 为 LLM 智能体配备了用于多尺度探索的图原生工具，通过搜索约束思维来规范工具使用以平衡准确性与效率，并采用图条件课程强化学习策略，在无需逐步监督的情况下稳定长程策略学习。在多种文本属性图（TAG）基准测试和多个 LLM 主干模型上，AgentGL 的表现显著优于强力的 GraphLLMs 和 GraphRAG 基准，在节点分类和链路预测中分别实现了高达 17.5% 和 28.4% 的绝对提升。这些结果表明，AGL 是使 LLM 能够在复杂关系环境中自主导航和推理的一个极具前景的前沿方向。代码已在 https://github.com/sunyuanfu/AgentGL 公开。

## Abstract
Large Language Models (LLMs) increasingly rely on agentic capabilities-iterative retrieval, tool use, and decision-making-to overcome the limits of static, parametric knowledge. Yet existing agentic frameworks treat external information as unstructured text and fail to leverage the topological dependencies inherent in real-world data. To bridge this gap, we introduce Agentic Graph Learning (AGL), a paradigm that reframes graph learning as an interleaved process of topology-aware navigation and LLM-based inference. Specifically, we propose AgentGL, the first reinforcement learning (RL)-driven framework for AGL. AgentGL equips an LLM agent with graph-native tools for multi-scale exploration, regulates tool usage via search-constrained thinking to balance accuracy and efficiency, and employs a graph-conditioned curriculum RL strategy to stabilize long-horizon policy learning without step-wise supervision. Across diverse Text-Attributed Graph (TAG) benchmarks and multiple LLM backbones, AgentGL substantially outperforms strong GraphLLMs and GraphRAG baselines, achieving absolute improvements of up to 17.5% in node classification and 28.4% in link prediction. These results demonstrate that AGL is a promising frontier for enabling LLMs to autonomously navigate and reason over complex relational environments. The code is publicly available at https://github.com/sunyuanfu/AgentGL.

*   **研究动机**：尽管大语言模型（LLM）在处理非结构化文本方面表现出色，但在处理具有复杂拓扑结构的**文本属性图（TAG）**时仍面临挑战。现有的方法（如 GraphLLM 或 GraphRAG）通常是静态的，无法在推理过程中根据需要动态、自主地探索图结构。
*   **核心问题**：如何构建一个能够像人类专家一样，在图环境中进行“拓扑感知导航”与“迭代推理”交替进行的智能体系统？
*   **整体含义**：论文提出了**智能体图学习（Agentic Graph Learning, AGL）**范式，将图学习重新定义为一个顺序决策过程，通过强化学习（RL）训练 LLM 智能体自主搜索证据并进行推理。

### 2. 方法论
AgentGL 框架由三个核心支柱组成：
*   **图原生搜索工具（GNS Tools）**：为智能体配备了四种工具以实现多尺度探索：
    *   **1-hop/2-hop 邻域搜索**：获取局部拓扑证据。
    *   **结构显著性搜索（SS）**：基于 PageRank 分数识别全局关键节点。
    *   **图稠密搜索（DENSE）**：基于语义向量检索相关节点，弥补结构断裂。
*   **两阶段强化学习策略**：
    *   **阶段 1：策略引导（Bootstrapping）**：通过奖励函数（准确性+格式+工具覆盖率）引导模型学会使用图工具。
    *   **阶段 2：缓解搜索过度（MSO）**：引入“搜索约束思维”，通过惩罚冗余搜索和奖励深度推理（认知密度正则化），让模型学会“多思少搜”，提高效率。
*   **图条件课程学习（GCCL）**：
    *   利用图的固有属性（如节点度数、同质性、语义相似度）量化任务难度，遵循“由易到难”的顺序训练，以稳定长程策略的收敛。
*   **算法实现**：采用了 GRPO 和 REINFORCE++ 两种主流的无评论员（Critic-free）策略优化算法。

### 3. 实验设计
*   **数据集**：涵盖 3 个领域的 7 个 TAG 数据集：引用网络（OGB-Arxiv, PubMed, Arxiv-2023）、亚马逊产品图（OGB-Products, Photo, Computers）和社交网络（Reddit）。
*   **任务场景**：节点分类（NC）和链路预测（LP），包括**域内评估**和**零样本迁移（Zero-shot）**设置。
*   **对比基准（13 种方法）**：

*   **硬件设施**：实验在一台配备 **8 张 NVIDIA H100-80G SXM5 GPU** 的节点上运行。
*   **软件框架**：使用 Open-RLHF 作为主要的强化学习训练框架。

### 5. 实验数量与充分性
*   **实验规模**：论文进行了大规模的对比实验，涵盖了不同参数规模（3B 和 7B）的背骨模型。
*   **消融实验**：针对两阶段训练、奖励项（rCOV, CDR, RTT）、课程学习（GCCL）以及搜索超参数（λr）均做了详细的消融分析。
*   **客观性与公平性**：所有基准模型均使用相同的 LLM 背骨，并为非图原生智能体提供了相同的图搜索空间，确保了对比的公正性。实验还包含了方差分析以验证 RL 训练的稳定性。

### 6. 主要结论与发现
*   **性能卓越**：AgentGL 在所有任务中均显著优于基准。在节点分类上最高提升 **17.5%**，在链路预测上最高提升 **28.4%**。
*   **迁移能力强**：在零样本迁移设置下，AgentGL 的领先优势扩大，证明了其学习到的是通用的图导航策略而非简单的模式记忆。
*   **效率与准确性的平衡**：通过第二阶段训练，模型在减少约 **17.5% - 22%** 搜索次数的同时，准确率反而有所提升。
*   **模型规模效应**：随着背骨模型从 3B 扩展到 7B，智能体的图学习和泛化能力显著增强。

### 7. 优点（亮点）
*   **范式创新**：首次将图学习系统性地建模为受 RL 驱动的智能体决策过程。
*   **工具设计完备**：结合了局部拓扑、全局结构和语义关联，覆盖了图探索的多个维度。
*   **训练策略科学**：引入课程学习解决图任务中样本难度差异大的问题，利用“搜索约束思维”解决了智能体常见的“复读机式检索”痛点。

### 8. 不足与局限
*   **模态限制**：目前仅支持文本属性图，尚未扩展到包含图像或视频的多模态属性图。
*   **训练复杂性**：两阶段 RL 训练对数据分配和超参数调节较为敏感，增加了部署难度。
*   **图密度挑战**：在极稠密的图中，如何进一步优化搜索效率和处理噪声仍需探索。
*   **推理成本**：尽管优化了搜索次数，但多轮交互推理的延迟和计算成本仍高于传统的静态 GraphLLM。
