## 摘要
智能体技能是程序性知识和可执行资源的结构化包，智能体在推理时动态加载这些技能，已成为增强大语言模型（LLM）智能体的一种可靠机制。然而，推理时的技能增强存在根本性的局限：检索噪声会引入无关的指导，注入的技能内容会带来巨大的 Token 开销，且模型从未真正习得它仅仅是在遵循的知识。我们探讨技能是否可以内化到模型参数中，从而在无需任何运行时技能检索的情况下实现零样本自主行为。我们提出了 SKILL0，这是一个专为技能内化设计的上下文强化学习框架。SKILL0 引入了一种训练阶段的课程学习方案，从完整的技能上下文开始并逐步将其撤出。技能在离线状态下按类别分组，并与交互历史一起渲染成紧凑的视觉上下文，教导模型进行工具调用和多轮任务完成。随后，动态课程会评估每个技能文件的同策略（on-policy）辅助作用，在呈线性衰减的预算内，仅保留当前策略仍能从中获益的技能，直到智能体在完全零样本的设置下运行。广泛的智能体实验表明，SKILL0 相比标准强化学习基线取得了显著提升（ALFWorld 提升 9.7%，Search-QA 提升 6.6%），同时保持了每步少于 0.5k Token 的高效上下文。我们的代码可在 https://github.com/ZJU-REAL/SkillZero 获取。

## Abstract
Agent skills, structured packages of procedural knowledge and executable resources that agents dynamically load at inference time, have become a reliable mechanism for augmenting LLM agents. Yet inference-time skill augmentation is fundamentally limited: retrieval noise introduces irrelevant guidance, injected skill content imposes substantial token overhead, and the model never truly acquires the knowledge it merely follows. We ask whether skills can instead be internalized into model parameters, enabling zero-shot autonomous behavior without any runtime skill retrieval. We introduce SKILL0, an in-context reinforcement learning framework designed for skill internalization. SKILL0 introduces a training-time curriculum that begins with full skill context and progressively withdraws it. Skills are grouped offline by category and rendered with interaction history into a compact visual context, teaching he model tool invocation and multi-turn task completion. A Dynamic Curriculum then evaluates each skill file's on-policy helpfulness, retaining only those from which the current policy still benefits within a linearly decaying budget, until the agent operates in a fully zero-shot setting. Extensive agentic experiments demonstrate that SKILL0 achieves substantial improvements over the standard RL baseline (+9.7\% for ALFWorld and +6.6\% for Search-QA), while maintaining a highly efficient context of fewer than 0.5k tokens per step. Our code is available at https://github.com/ZJU-REAL/SkillZero.

### 1. 核心问题与研究动机
*   **核心问题**：当前的 LLM 智能体主要依赖“推理时技能增强”（Inference-time Skill Augmentation），即从技能库检索相关指令并放入 Prompt 中。这种方式存在三大缺陷：
    1.  **检索噪声**：引入无关指导干扰决策。
    2.  **Token 开销**：长上下文导致推理成本高、速度慢。
    3.  **缺乏真知**：模型只是在“遵循”指令而非“掌握”技能，知识存在于上下文中而非参数中。
*   **研究动机**：借鉴人类学习过程（从依赖说明书到形成肌肉记忆），探索能否在训练时利用技能引导，但在推理时实现“零检索”的自主行为。

### 2. 方法论
SKILL0 提出了**上下文智能体强化学习（In-Context Agentic RL）**，核心思想是“训练时有技能，推理时零技能”。
*   **视觉上下文渲染（Context Rendering）**：为了降低 Token 开销，将文本历史和技能描述渲染成紧凑的 RGB 图像，通过视觉编码器压缩。模型可自适应生成压缩率 $c_t$。
*   **上下文强化学习（ICRL）**：在 RL 训练的 Rollout 阶段提供技能引导，但在推理阶段完全移除。通过复合奖励函数（任务成功 + 压缩效率）进行优化。
*   **动态课程学习（Dynamic Curriculum）**：
    *   **离线技能分组**：按类别组织技能文件。
    *   **辅助性评估（Helpfulness Evaluation）**：定义指标 $\Delta k$（有技能时的准确率减去无技能时的准确率），评估当前策略对某项技能的依赖程度。
    *   **线性预算衰减**：设定技能预算 $M$，随训练阶段线性减少。
    *   **过滤与排序**：动态保留 $\Delta k > 0$ 且排名靠前的技能，直到预算归零，迫使模型将外部知识转化为内在策略。

### 3. 实验设计
*   **实验场景/数据集**：
    *   **ALFWorld**：具身智能文本游戏，包含 6 类家庭任务（如清洁、加热、放置等）。

    *   **技能/经验相关方法**：EvolveR、SkillRL（推理时需检索技能的强基线）、ExpeL、Mem0 等。

*   **实验规模**：在两个大类任务（具身导航与复杂问答）下的十余个子数据集上进行了测试。
*   **消融实验**：针对技能预算 $M$ 的衰减策略、动态课程的必要性（过滤与排序的作用）、验证间隔 $d$ 等进行了详细消融。
*   **充分性评价**：实验设计较为全面，不仅对比了最终成功率，还详细展示了训练过程中的奖励曲线、技能辅助性变化曲线以及 Token 消耗对比，客观地证明了“内化”过程的发生。

### 6. 主要结论与发现
*   **性能提升**：在推理时不使用任何技能检索的情况下，SKILL0 在 ALFWorld 上比标准 RL 基线提升了 **9.7%**，在 Search-QA 上提升了 **6.6%**。
*   **高效推理**：由于无需注入技能文本且使用了视觉压缩，每步推理的 Token 消耗控制在 **0.5k 以下**，比 SkillRL 等方法降低了 5 倍以上。
*   **内化趋势**：实验观察到技能的“辅助性”呈现先升后降的趋势，证明模型早期利用技能探索，后期通过内化摆脱了对技能的依赖。

### 7. 优点
*   **范式创新**：首次将技能内化明确定义为强化学习的训练目标，解决了智能体对 Prompt 的过度依赖。
*   **极高的推理效率**：通过内化和视觉压缩双重手段，极大地优化了长程任务的 Token 成本。
*   **自适应性**：动态课程学习避免了硬编码的训练计划，能根据模型当前的掌握程度自动调整技能撤减节奏。

### 8. 不足与局限
*   **初始依赖**：SKILL0 的效果高度依赖于初始技能库（SkillBank）的质量，如果初始技能有误，可能会引导错误的内化方向。
*   **离线成本**：技能分组和验证子任务的划分需要一定的先验知识和离线处理工作。
*   **泛化边界**：虽然在域外数据集（如 Bamboogle）表现良好，但对于完全陌生的新技能领域，可能仍需重新经历内化训练过程。
