## 摘要
最近，潜空间推理（latent reasoning）被引入大语言模型（LLMs）中，以利用连续空间内的丰富信息。然而，由于缺乏随机采样，这些方法不可避免地会坍缩为确定性推理，从而无法发现多样化的推理路径。为了弥补这一差距，我们通过 Gumbel-Softmax 向潜空间推理中注入了可控的随机性，恢复了大语言模型的探索能力，并增强了其与强化学习（RL）的兼容性。在此基础上，我们提出了潜空间推理策略优化（Latent Reasoning Policy Optimization，简称 LEPO），这是一个直接将强化学习应用于连续潜表示的新型框架。具体而言，在展开（rollout）阶段，LEPO 保持随机性以实现多样化的轨迹采样；而在优化阶段，LEPO 为潜表示和离散标记（tokens）构建了统一的梯度估计。大量实验表明，LEPO 在离散推理和潜空间推理方面均显著优于现有的强化学习方法。

## Abstract
Recently, latent reasoning has been introduced into large language models (LLMs) to leverage rich information within a continuous space. However, without stochastic sampling, these methods inevitably collapse to deterministic inference, failing to discover diverse reasoning paths. To bridge the gap, we inject controllable stochasticity into latent reasoning via Gumbel-Softmax, restoring LLMs' exploratory capacity and enhancing their compatibility with Reinforcement Learning (RL). Building on this, we propose \textbf{\underline{L}}atent R\textbf{\underline{e}}asoning \textbf{\underline{P}}olicy \textbf{\underline{O}}ptimization~(\textbf{LEPO}), a novel framework that applies RL directly to continuous latent representations. Specifically, in rollout stage, LEPO maintains stochasticity to enable diverse trajectory sampling, while in optimization stage, LEPO constructs a unified gradient estimation for both latent representations and discrete tokens. Extensive experiments show that LEPO significantly outperforms existing RL methods for discrete and latent reasoning.

### 1. 核心问题与研究动机
*   **核心问题**：现有的潜空间推理（Latent Reasoning）方法虽然利用了连续空间的丰富信息，但大多是**确定性推理**，缺乏随机采样机制。
*   **研究动机**：在强化学习（RL）中，探索（Exploration）至关重要。确定性推理会导致推理路径坍缩，无法发现多样化的解题路径。论文旨在为潜空间推理引入可控的随机性，从而恢复模型的探索能力，并构建一个能直接优化连续表示的强化学习框架。

### 2. 方法论：LEPO 框架
LEPO 的核心思想是将潜空间推理建模为一个可学习的策略，并直接在连续表示上进行策略优化。
*   **随机采样（Gumbel-Softmax）**：在 Rollout 阶段，不直接取概率最大的 Token，而是通过 Gumbel-Softmax 对词表分布 $\pi_t$ 注入噪声，生成随机潜 Token $z_t$。这既保留了分布信息，又实现了轨迹级的多样化探索。
*   **混合推理轨迹**：模型先生成 $L_g$ 个随机潜 Token 进行“思考”，随后切换到离散采样模式生成可读的最终答案。
*   **统一优化目标**：
    *   **离散部分**：对答案部分的 Token 使用标准的 REINFORCE 算法。
    *   **潜空间部分**：提出“软代理目标”（Soft Surrogate Objective），将采样的随机向量 $z_t$ 视为“软标签”，最大化当前策略与该标签的对齐度，并由奖励产生的优势值（Advantage）进行加权。
*   **梯度估计**：通过统一的公式对潜推理步骤和离散答案步骤进行联合梯度上升，确保整个推理链条都朝着高奖励方向优化。

### 6. 主要结论与发现
*   **性能领先**：LEPO 在所有基准测试中均显著优于现有的离散 RL（如 GRPO）和潜空间 RL 方法。在 Qwen2.5-7B 上，平均 Pass@1 提升了 1.76%，Pass@32 提升了 1.95%。
*   **探索能力强**：引入 Gumbel 噪声后，模型在训练过程中保持了更高的熵，能够将“不可解”的难题转化为“中等难度”的可解题。
*   **Token 效率高**：相比离散推理，LEPO 使用更短的潜表示即可达到更好的效果，显著降低了推理时的 Token 消耗。
*   **收敛性好**：实验显示 LEPO 的训练奖励曲线比使用高斯噪声或离散 RL 的方法收敛更快、更稳。

### 7. 优点与亮点
*   **创新性**：首次将 Gumbel-Softmax 随机性引入潜空间推理并结合 RL 策略优化，解决了潜空间推理的“坍缩”问题。
*   **统一性**：构建了一个能够同时处理“软”潜 Token 和“硬”离散 Token 的统一梯度估计框架。
*   **实用性**：在提升推理能力的同时，通过减少 Token 数量提高了推理效率。

### 8. 不足与局限
*   **可解释性**：潜空间推理步骤对人类来说是不可读的（虽然可以通过解码 Top-1 Token 近似理解），这在需要高度透明推理过程的场景中可能受限。
*   **生成范式限制**：目前采用“先潜推理、后离散答案”的顺序结构，尚不支持潜 Token 与离散 Token 的动态交替（即边写边想）。
*   **任务范围**：主要集中在数学和逻辑推理任务，在更广泛的创意写作或开放式对话任务中的表现尚待验证。
