## 摘要
大语言模型（LLMs）越来越多地依赖于智能体能力（如迭代检索、工具使用和决策）来克服静态参数化知识的局限性。然而，现有的智能体框架将外部信息视为非结构化文本，未能利用现实世界数据中固有的拓扑依赖关系。为了填补这一空白，我们引入了智能体图学习（Agentic Graph Learning, AGL），这是一种将图学习重新定义为拓扑感知导航与基于 LLM 推理交替进行的范式。具体而言，我们提出了 AgentGL，这是首个用于 AGL 的强化学习（RL）驱动框架。AgentGL 为 LLM 智能体配备了用于多尺度探索的图原生工具，通过搜索约束思维来规范工具使用以平衡准确性与效率，并采用图条件课程强化学习策略，在无需逐步监督的情况下稳定长程策略学习。在多种文本属性图（TAG）基准测试和多个 LLM 主干模型上，AgentGL 的表现显著优于强力的 GraphLLMs 和 GraphRAG 基准，在节点分类和链路预测中分别实现了高达 17.5% 和 28.4% 的绝对提升。这些结果表明，AGL 是使 LLM 能够在复杂关系环境中自主导航和推理的一个极具前景的前沿方向。代码已在 https://github.com/sunyuanfu/AgentGL 公开。

## Abstract
Large Language Models (LLMs) increasingly rely on agentic capabilities-iterative retrieval, tool use, and decision-making-to overcome the limits of static, parametric knowledge. Yet existing agentic frameworks treat external information as unstructured text and fail to leverage the topological dependencies inherent in real-world data. To bridge this gap, we introduce Agentic Graph Learning (AGL), a paradigm that reframes graph learning as an interleaved process of topology-aware navigation and LLM-based inference. Specifically, we propose AgentGL, the first reinforcement learning (RL)-driven framework for AGL. AgentGL equips an LLM agent with graph-native tools for multi-scale exploration, regulates tool usage via search-constrained thinking to balance accuracy and efficiency, and employs a graph-conditioned curriculum RL strategy to stabilize long-horizon policy learning without step-wise supervision. Across diverse Text-Attributed Graph (TAG) benchmarks and multiple LLM backbones, AgentGL substantially outperforms strong GraphLLMs and GraphRAG baselines, achieving absolute improvements of up to 17.5% in node classification and 28.4% in link prediction. These results demonstrate that AGL is a promising frontier for enabling LLMs to autonomously navigate and reason over complex relational environments. The code is publicly available at https://github.com/sunyuanfu/AgentGL.

*   **核心问题**：如何构建一个能够像人类专家一样，在图结构中自主导航、迭代检索证据并进行拓扑感知推理的智能体系统？
*   **整体含义**：论文提出了**智能体图学习（Agentic Graph Learning, AGL）**范式，将图学习重新定义为“拓扑感知导航”与“LLM 推理”交替进行的决策过程，并开发了首个由强化学习（RL）驱动的框架 **AgentGL**。

### 2. 方法论
AgentGL 框架由三个核心支柱组成：
*   **图原生搜索工具（GNS Tools）**：为 LLM 提供了一套多尺度的图操作原语，包括：
    *   **1-hop/2-hop 邻域搜索**：获取局部拓扑证据。
    *   **结构显著性搜索（SS）**：基于 PageRank 等结构先验识别全局关键节点。
    *   **图稠密搜索（Dense）**：利用语义向量相似度跨越拓扑孤岛寻找关联。
*   **两阶段强化学习策略**：
    *   **阶段 1：策略引导（Bootstrapping）**：通过奖励函数（准确性 + 格式 + 工具覆盖率）训练模型掌握工具调用的基本能力。
    *   **阶段 2：缓解搜索过度（MSO）**：引入“搜索约束思维”，通过回顾性终止触发器和认知密度正则化，惩罚冗余搜索，鼓励模型在有限证据下进行深度推理。
*   **图条件课程学习（GCCL）**：
    *   利用图的固有属性（如节点度、同质性、语义相似度）量化任务难度。
    *   训练过程遵循“由易到难”的顺序，以稳定长程策略的收敛。
*   **算法实现**：支持 GRPO 和 REINFORCE++ 等主流强化学习算法，将 AGL 建模为“推理-行动-观察”的循环。

### 6. 主要结论与发现
*   **性能突破**：AgentGL 在节点分类上最高提升 **17.5%**，在链路预测上最高提升 **28.4%**。
*   **动态优于静态**：交替进行的“搜索-推理”循环显著优于静态的上下文注入（GraphLLM）或简单的检索增强（GraphRAG）。
*   **效率平衡**：通过第二阶段的 MSO 训练，模型在减少约 **17.5%** 工具调用的同时，准确率反而有所提升，证明了“少即是多”的搜索策略有效性。
*   **泛化能力**：AgentGL 在未见过的图领域表现出极强的零样本迁移能力，远超静态模型。

### 7. 优点
*   **范式创新**：首次将图学习定义为受约束的智能体决策过程，并引入 RL 进行端到端优化。
*   **工具设计完备**：结合了局部拓扑、全局结构和语义关联，覆盖了图探索的多个维度。
*   **训练稳定性**：通过课程学习（GCCL）解决了长程策略在无步骤监督下难以收敛的问题。

### 8. 不足与局限
*   **模态限制**：目前仅支持文本属性图，尚未扩展到包含图像、视频等多模态属性的图数据。
*   **计算开销**：虽然 MSO 阶段优化了推理效率，但 RL 训练过程（尤其是多轮 Rollout）对算力需求较高。
*   **图密度敏感性**：在极度稠密的图中，如何进一步优化搜索路径以避免信息过载仍有待探索。
*   **数据分配**：两阶段训练之间的数据分配比例对最终性能有一定影响，需精细调优。
