## 摘要
自适应交通信号控制（ATSC）旨在通过实时调整红绿灯来优化交通流并最大限度地减少延迟。多智能体强化学习（MARL）的最新进展在 ATSC 方面展现出潜力，但现有方法仍受限于表征能力不足，在复杂多变的交通环境中往往导致次优性能和较差的泛化能力。另一方面，大语言模型（LLMs）在语义表征、推理和分析方面表现出色，但其幻觉倾向和缓慢的推理速度往往阻碍了它们在决策任务中的直接应用。为了应对这些挑战，我们提出了一种名为 LATS 的新型学习范式，它整合了 LLMs 和 MARL，利用前者的强大先验知识和归纳能力来增强后者的决策过程。具体而言，我们引入了一个即插即用的教师-学生学习模块，其中经过训练的嵌入式 LLM 作为教师，生成捕捉每个路口拓扑结构和交通动态的丰富语义特征。随后，一个更简单的（学生）神经网络通过潜在空间中的知识蒸馏来学习模拟这些特征，使最终模型能够独立于 LLM 运行，用于强化学习决策过程的下游应用。这种整合显著增强了模型在不同交通场景下的整体表征能力，从而带来了更高效且泛化性更强的控制策略。在多种交通数据集上的广泛实验表明，我们的方法增强了强化学习模型的表征学习能力，从而在整体性能和泛化能力上均优于传统的强化学习和仅使用 LLM 的方法。

## Abstract
Adaptive Traffic Signal Control (ATSC) aims to optimize traffic flow and minimize delays by adjusting traffic lights in real time. Recent advances in Multi-agent Reinforcement Learning (MARL) have shown promise for ATSC, yet existing approaches still suffer from limited representational capacity, often leading to suboptimal performance and poor generalization in complex and dynamic traffic environments. On the other hand, Large Language Models (LLMs) excel at semantic representation, reasoning, and analysis, yet their propensity for hallucination and slow inference speeds often hinder their direct application to decision-making tasks. To address these challenges, we propose a novel learning paradigm named LATS that integrates LLMs and MARL, leveraging the former's strong prior knowledge and inductive abilities to enhance the latter's decision-making process. Specifically, we introduce a plug-and-play teacher-student learning module, where a trained embedding LLM serves as a teacher to generate rich semantic features that capture each intersection's topology structures and traffic dynamics. A much simpler (student) neural network then learns to emulate these features through knowledge distillation in the latent space, enabling the final model to operate independently from the LLM for downstream use in the RL decision-making process. This integration significantly enhances the overall model's representational capacity across diverse traffic scenarios, thus leading to more efficient and generalizable control strategies. Extensive experiments across diverse traffic datasets empirically demonstrate that our method enhances the representation learning capability of RL models, thereby leading to improved overall performance and generalization over both traditional RL and LLM-only approaches. ### 1. 核心问题与研究动机
*   **核心问题**：如何在保证实时决策效率的同时，提升多智能体强化学习（MARL）在复杂动态交通信号控制（ATSC）任务中的表征能力和泛化性能。
*   **研究动机**：
    *   **MARL 的局限性**：现有的强化学习模型通常参数量较小，表征能力有限，难以处理具有多样化拓扑结构和动态交通需求的复杂路网。
    *   **LLM 的优缺点**：大语言模型（LLM）具备极强的语义理解和推理能力，但直接用于实时决策时存在“幻觉”风险、推理速度慢且计算开销巨大。
    *   **研究目标**：结合 LLM 的先验知识与 MARL 的决策效率，提出一种既能利用 LLM 语义特征又能在部署时脱离 LLM 的新框架。

### 2. 方法论
LATS 框架采用**教师-学生（Teacher-Student）学习范式**，核心思想是将 LLM 作为“知识教师”，通过蒸馏技术增强“学生”强化学习模型的特征表达。
*   **教师模块（训练阶段）**：
    *   **提示词构建**：将路口的静态拓扑（车道数、限速等）和动态交通状态（车辆数、排队长度等）转化为结构化文本提示词。
    *   **语义嵌入生成**：使用预训练的嵌入式 LLM（如 `jina-embeddings-v2-small-en`）将文本转化为高维语义向量。
*   **学生模块与知识蒸馏**：
    *   **潜在空间对齐**：引入两个变分自编码器（VAE），分别对 LLM 的语义特征和 RL 的状态向量进行编码。
    *   **KL 散度优化**：通过最小化两个潜在分布之间的 KL 散度，使简单的学生网络能够模拟 LLM 生成的复杂语义特征。
*   **强化学习决策**：
    *   基于 PPO 算法，将学生网络提取的潜在语义特征与原始交通状态融合，用于策略（Policy）和价值（Value）函数的计算。
*   **部署阶段**：仅保留学生网络和 RL 策略，实现 **LLM-free（无 LLM）推理**，确保毫秒级响应。

### 3. 实验设计
*   **数据集与场景**：
    *   **Grid 5×5**：包含 25 个相同路口的同质化网格路网。
    *   **Monaco**：基于摩纳哥真实地图的异质化路网，包含 28 个具有不同拓扑结构的路口。
*   **Benchmark 与对比方法**：

    *   **纯 RL 方法**：独立学习（MA2C, IQL）和参数共享模型（IPPO, AttendLight, CoLight, HeteroLight）。
*   **评估指标**：平均排队长度、平均车速、路口延迟、完成率、行程时间及行程延迟。

*   **实验规模**：涵盖了同质和异质路网的性能对比、**零样本迁移（Zero-shot Transfer）**实验（在摩纳哥训练，在 Grid 测试）、以及针对不同交通需求（低、中、高）的泛化性测试。

### 6. 主要结论
*   **性能领先**：LATS 在所有关键交通指标上均显著优于传统 RL 和纯 LLM 方法。
*   **泛化性强**：通过 LLM 提供的语义先验，模型在未见过的路网和交通需求下表现出极强的零样本迁移能力。
*   **效率平衡**：通过师生蒸馏，成功在保持 RL 高速推理优势的同时，获得了 LLM 级别的语义理解能力。
*   **即插即用**：该模块可以集成到现有的多种通用 RL 交通控制框架中（如 AttendLight），并带来性能提升。

### 7. 优点与亮点
*   **架构创新**：首次将 LLM 的语义表征能力通过潜在空间蒸馏引入到 RL 交通控制中，而非简单的提示词工程。
*   **实用性高**：解决了 LLM 落地实时控制系统的推理延迟和成本问题，实现了“训练时有 LLM，运行时无 LLM”。
*   **多模态融合**：将结构化的交通向量数据转化为非结构化的文本语义，增强了模型对路口拓扑差异的感知。

### 8. 不足与局限
*   **训练开销**：由于训练过程中需要频繁调用 LLM 生成嵌入向量，训练时间比传统 RL 高出数倍。
*   **提示词依赖**：目前的语义生成依赖于预定义的固定模板，可能无法完全捕捉极端或罕见交通状况的细微差别。
*   **缺乏智能体间协作语义**：研究重点在于提升单个路口的表征，尚未深入探讨如何利用 LLM 生成路口间的协作通信语义。
*   **环境局限**：实验主要在仿真环境（SUMO）中进行，尚未在包含天气、事故等更复杂现实干扰因素的场景中验证。
